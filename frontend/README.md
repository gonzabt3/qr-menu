# QR Menu Admin Frontend

This is the Next.js frontend for the QR Menu admin backoffice.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Set up environment variables:

```bash
cp .env.local.example .env.local
```

Edit `.env.local` and configure:

```env
NEXT_PUBLIC_BACKEND_URL=http://localhost:3000
```

3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001/admin) with your browser to access the admin interface.

## Documentation

For detailed documentation about the admin backoffice, see [../docs/Admin.md](../docs/Admin.md).

## Build

```bash
npm run build
npm start
```
